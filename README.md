# Unsupervised Learning with Scikit-learn

## Description

This project explores fundamental unsupervised learning techniques using Scikit-learn. The objective is to preprocess tabular data, reduce dimensionality, and discover hidden patterns through clustering algorithms. The project demonstrates the complete workflow of unsupervised machine learning, from feature scaling and Principal Component Analysis (PCA) to clustering with K-Means and Agglomerative Hierarchical Clustering.

## Tasks Covered

* Feature Standardization using StandardScaler
* Dimensionality Reduction with PCA
* K-Means Clustering
* Optimal Cluster Selection using Inertia and Silhouette Scores
* Agglomerative Hierarchical Clustering

## Technologies Used

* Python
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Conclusion

This project provides practical experience with essential unsupervised learning techniques. By combining preprocessing, dimensionality reduction, and clustering methods, it demonstrates how machine learning can uncover meaningful structures and relationships within unlabeled data.
